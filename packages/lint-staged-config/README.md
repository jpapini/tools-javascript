## How to install

Run the following command to install the required dependencies:

```shell
$ <yarn|npm|pnpm> add -D \
    @jpapini/lint-staged-config \
    lint-staged \
    eslint \
    prettier \
    sort-package-json@1.51.0 \
    npm-package-json-lint
```
